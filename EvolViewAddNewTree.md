## Add a new tree to a new project ##

  * at the "login" page, click "Use without an account";
> > ![https://www.evernote.com/shard/s130/sh/ced8b4fa-8a54-4959-bec6-b9575f692efe/f31e487236749603963bfe66c34e2ce5/res/9eadb4df-dbf7-42bb-902c-8e5161d7d291/skitch.png](https://www.evernote.com/shard/s130/sh/ced8b4fa-8a54-4959-bec6-b9575f692efe/f31e487236749603963bfe66c34e2ce5/res/9eadb4df-dbf7-42bb-902c-8e5161d7d291/skitch.png)

  * at the "app" page, click the folder icon on the "Basic" tab (the only icon that's clickable if no tree is active). in the popup window, enter the following contents and click the button "Submit" at the bottom:
```
    Project: new project
    Name: test tree
    Data: (chicken,((mouse,rat),(chimp,human)));
```
